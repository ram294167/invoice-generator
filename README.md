# Invoice Generator

A modern React-powered invoice builder that creates professional PDFs. Migrated from static HTML to dynamic React for better usability.

## ✨ Key Features

- **Dynamic Editing**: Real-time invoice customization with company details, dates, and event info
- **Smart Calculations**: Automatic GST, discounts, and totals with live updates
- **File Uploads**: Add logos, QR codes, and signatures with instant preview
- **PDF Export**: One-click professional PDF generation using html2canvas + jsPDF
- **Responsive**: Works perfectly on desktop and mobile
- **Data Persistence**: Saves work locally in browser

## 🛠️ Tech Stack

- **Frontend**: React, Vite, CSS
- **PDF Generation**: html2canvas, jsPDF
- **Storage**: Browser LocalStorage

## 🚀 Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/your-username/invoice-generator.git
   cd invoice-generator
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:5173`

3. **Or Use Online**: [Live Demo](https://invoice-generator-hioi.vercel.app/)

## 📖 How to Use

1. Fill invoice details (company, date, event)
2. Add items with quantity, rate, discount, GST
3. Upload logos/signatures if needed
4. Preview invoice in real-time
5. Download as PDF

## 🏗️ Build for Production
